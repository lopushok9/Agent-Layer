# market-pulse                                                                                                                                                    
                                                                                                                                                                    
  Real-time market analytics skill for OpenClaw.                                                                                                                    
                                                                                                                                                                    
  ## Install                                                                                                                                                        
                                                                                                                                                                    
npx clawhub@latest install market-pulse                                                                                                         
                                                                                                                                                                    
  Features                                                                                                                                                          
                                                                                                                                                                    
  - Crypto prices (BTC, ETH, SOL...)                                                                                                                                
  - Fear & Greed Index                                                                                                                                              
  - Top DeFi yields                                                                                                                                                 
  - Upcoming crypto events                                                                                                                                          
  - Stock indices (S&P 500, NASDAQ)                                                                                                                                 
                                                                                                                                                                    
  Usage                                                                                                                                                             
                                                                                                                                                                                                                                                                                                     
  - "BTC price?" — quick price check                                                                                                                                
  - "Top DeFi yields" — best APY rates
  - etc.
