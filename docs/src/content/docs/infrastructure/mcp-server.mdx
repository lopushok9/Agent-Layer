# MCP Server

`mcp-server/` is the finance and crypto MCP layer in AgentLayer.

It is used for structured read-oriented access to:

- prices
- DeFi data
- wallet and token data
- gas data
- ERC-8004 agent discovery

It complements the wallet stack rather than replacing it.
